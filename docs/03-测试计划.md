## 4. 准确率评估

### 4.1 评估方法

| 步骤 | 说明 |
|------|------|
| 1. 准备测试集 | 100 条常见 HR 问题 + 标准答案 |
| 2. 盲测 | 不告诉模型标准答案，让其回答 |
| 3. 人工评估 | HR 专家逐条评估回答是否正确 |
| 4. 统计 | 计算准确率 = 正确数 / 总数 |

### 4.2 评估标准

| 准确率 | 评级 | 处理 |
|--------|------|------|
| ≥ 95% | 优秀 | 可直接上线 |
| 90%-95% | 良好 | 优化后可上线 |
| 80%-90% | 一般 | 需要改进 |
| < 80% | 不合格 | 不可上线 |

### 4.3 评估问题清单（示例）

| 序号 | 问题 | 预期回答要点 |
|------|------|--------------|
| 1 | 年假有多少天？ | 根据入职年限，5-15天 |
| 2 | 婚假有几天？ | 根据婚姻状况，通常 10-30 天 |
| 3 | 加班有加班费吗？ | 加班调休或加班费政策 |
| 4 | 社保怎么缴纳？ | 缴纳比例、基数 |
| 5 | 离职流程是什么？ | 提前通知、交接、离职证明 |
| ... | ... | ... |

## 6. 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| 测试数据不足 | 准确率评估不准 | 扩大测试问题集 |
| 环境不稳定 | 测试结果不可靠 | 使用独立测试环境 |
| 性能瓶颈 | 上线后响应慢 | 提前压测，预留容量 |
| LLM 幻觉 | 回答不准确 | 增加 RAG 约束 + 人工抽检 |
